## 记录规则

1. 每个人用自己的名字/昵称/github命名文件夹
2. 个人简介和个人github地址写到个人文件夹的README.md里
3. 其余规则可自由指定，例如，以月份为名称，例：2018-03.md(每月一个md文件)记录每一天的学习情况，或以每日为单位，或以每一个主题为单位，如"vue.js"，然后每日update即可。
4. 以上规则先试用一个月，看看情况。
