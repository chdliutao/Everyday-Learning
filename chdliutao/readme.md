* 2018-03-21

```
  晚点名的开发由于速度以及效率原因结束，目前功能基本实现。
  我在本次开发中负责后台的界面制作，用Dophinphp的Zbuilder这样一个快速构造器，虽然本身的开发是建立在老师的
原有基础上，但是中间仍然遇到过很多的麻烦和问题，但好在在老师的帮助下捋清了基本的逻辑和基本界面的构成。
  总结这次开发晚点名的收获：
    1、首先感谢杨老师可以带着我，引领我学习。
    2、在开发过程中，虽然会有手册的指引，但是很多情况下依然需要自己想办法解决一些问题，可能手册中不会涉及，
但是遇到问题就要想办法解决而不是逃避，否则怎么能开发出好的产品。
    3、在开发过程中要设想好各种可能发生的情况，防止出现某些用户的不当操作使产品出现问题，这些在开发的时候都
应考虑进去。
    4、开发要注意效率是和时间的把控，在现在的社会，谁先谁快谁就占得了先机，因而也算是一个收获吧，以后的时间
尽可能提高开发的效率，将更多的时间投入到开发中。
    5、不得不承认，看到别人在用自己参与制作的应用中确实很开心很激动，当有人反馈它不好时也会有些难过，总之
这也才是第一次的开发经历，学到了好多知识，对thinkPHP的框架有了更好的理解，考虑到一些逻辑层的问题。
```
